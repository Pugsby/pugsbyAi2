Pugsby AIbeta | Create Character

Import

If you have a character in tavernai format, you can import it here. Images won't work, just json files.

I will soon add the ability to import from images.

Metadata

These are things that the bot will not be able to access, but it's visible to the user in some places.

The bot will NOT be able to access these*.

Image

The image will be used as the bot's profile picture.

Due to limitations of the tavernai format, you have to add the image yourself, assuming you imported it from json.

If you want to use a image from your computer, you can either upload it to a image sharing website or use Image to Data:URI.

Image can be anything, it will be previewed here:

Name

The name of the bot.

* The bot WILL be able to access this IF there is no set Chat Name.

Persona

This is what the bot will be able to access, this is not visible to the user with some exceptions.

Chat Name

This is what the bot will think their name is, this is what will displayed in chat.

* Users can see this.

Introduction

This is the first message the bot will send to the user.

* Users can see this.

Character Persona

Also known as Description to TavernAI or Long Description to character.ai. This is the mind and heart of your character, this is who your character is.

Summary

In a few words, describe your character. Also known as Short Description to character.ai.

Example Mesesages

Examples of what your character would say to the user, also known as Definition to character.ai, or Examples of Dialogue to Tavernai.

Scenario

This is the scenario the bot will be in, this is what the bot will do.

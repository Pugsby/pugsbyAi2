Pugsby AIbeta

PugsbyAI is in early beta, the home page is very basic and is just a list of characters.

Also, don't get excited when chatting, your chats wont be saved (yet)

In a future update, you will be able to save your chats and load them later, you will also be able to make custom user personas.

If you want to create user personas now, you can fork this repository and create your own personas.

By default, you won't see any characters, make them yourself here.
